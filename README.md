# 🤖 Gemini AI Clone

A modern **Gemini AI Clone** built using **React, Vite, and the Google Gemini API**. This project replicates the core chat experience of Gemini while adding custom features like Dark Mode and animated sidebar interactions.

## 🌐 Live Demo

🔗 https://gemini-ai-clone-delta.vercel.app

## 📂 GitHub Repository

🔗 https://github.com/rajkirankadali/Gemini-AI-Clone

---

## ✨ Features

- 🤖 Google Gemini API Integration
- 💬 AI Chat Interface
- 🌙 Dark Mode (Built Independently)
- 📂 Sidebar Collapse & Expand Animation
- 📝 Chat History
- ⚡ Typing Animation
- 📱 Fully Responsive Design
- ⚛️ React Context API for State Management
- 🚀 Deployed on Vercel

##  Installation

Clone the repository:

```bash
git clone https://github.com/rajkirankadali/Gemini-AI-Clone.git
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

Run the development server:

```bash
npm run dev
```
